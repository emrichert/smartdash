# smartdash
This smartdashboard visualizes county level COVID-19 cases in the US in 2020. A chloropleth map clearly shows the distribution of cases normalized to show case rates between counties with different populations. More specificially, the chloropleth map allows the user to hover over a county to get the name of the county (and its state) with their respective Covid-19 rate (out of 1000 people). The 5 classes are natural breaks and ArcGIS was used to attain the specific break values.
